# SigNoz-Span-Tree-Python
Span Tree Structure of a Trace in SigNoz

## Prerequisites

- Python 3.x installed on your system.

## How to run 

- ```cd``` into the ```SigNoz-Span-Tree-Python``` directory

- Run ```python3 main.py``` in your terminal
